# Memory Index

> Índice da memória de longo prazo (uma linha por memória, sem conteúdo aqui). Cada memória
> é um arquivo em `memory/` com frontmatter e tipo: user | feedback | project | reference.
> Ao criar uma memória, adicionar aqui uma linha: `- [Título](arquivo.md) — gancho curto`.
>
> **Teto: máximo 60 linhas.** Este índice é carregado em TODA sessão — sem teto, ele vira
> um segundo contexto inchado. Para entrar linha nova, alguma sai: apagar a obsoleta,
> fundir duplicadas ou promover a lição ao CLAUDE.md (deixando aqui só 1 linha-ponteiro).
